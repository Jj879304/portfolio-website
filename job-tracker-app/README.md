# 🚀 Job Tracker App

A full-stack job tracking application built with React, Node.js, Express, and SQLite.

Helps users track job applications and manage job status efficiently.

## ✨ Features

- Add jobs
- View jobs
- Change status
- Delete jobs
- SQLite database integration

## 🛠 Tech Stack

### Frontend
- React.js
- CSS

### Backend
- Node.js
- Express.js
- SQLite
